Current contents of this directory
----------------------------------
The files presently shipped here are NOT the final Kenney GLBs. They are
small placeholder GLBs generated programmatically via three.js GLTFExporter
(see ../../../../scripts/generate-placeholder-glbs.mjs) so that the Tier 2
asset pipeline (useGLTF + <Clone> + preload + fallback) is wired end-to-end
without a network fetch at build time.

The placeholders are released into the public domain (CC0) and may be
replaced by real Kenney GLBs at any time. The swap is mechanical: drop
matching-named .glb files into this directory and nothing else changes.
